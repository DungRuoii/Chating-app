### Setup .env file

```js
MONGODB_URI=...
PORT=5001
JWT_SECRET=...

CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...

NODE_ENV=development
```

### Build the app at the root

```shell
npm run build
```

### Start in backend
```shell
npm run start
```

### Start in frontend
```shell
npm run dev
```
